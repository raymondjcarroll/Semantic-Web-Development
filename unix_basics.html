UNIX Basics

Welcome to Unix Systems Administration

Welcome to your journey as a UNIX systems administrator. UNIX is an amazing muiltiuser operating system that's been at the forefront of computing since the early 1970's. UNIX was originally developed by AT&T's Bell Labs.

We still see UNIX in ways that you wouldn't expect. I'm sure that you have heard of Linux, a very popular UNIX-like OS, but other OSes are also based on UNIX. For example, macOS is based of NeXTSTEP, which is strongly based on UNIX. The Berkeley Software Distribution (BSD) family of operating systems are also based on UNIX. Even Microsoft Windows shares a lot of commonalities with UNIX.

What is a System Administrator?

What is a system administrator? It's actually a loaded question. Part of the reason that is a loaded question is that there is not a one-size-fits-all answer. There are some basic components of systems administration that may be a part of a system administrators daily duties:

  • Managing Backups
  • Managing Users
  • Answering Questions from end users
  • Managing System Updates
  • Installing Applications
  • Managing Networks

All of these functions are very important to the operation of a fleet of computers.
