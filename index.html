User Management in UNIX

As with any OS, it is important to have serveral users on a device, this allows for easy management for a system.

Creating User Accounts

Before creating a user group, you first must have infomation that will be needed to create the account

You will need:

  • A username (also called a login name) that will be assigned to the user.
  • A password to assign the account (in best practice, this will be changed on first sign in).
  • Any groups that you will add the user to.

In the FreeBSD OS, you may run adduser to create a user in a more interactive method. If you need anymore information, please run man adduser for more information.

Managing User Groups in UNIX

Creating groups of users is very important to system Administration.

UNIX systems make this very easy to manage. The pw utility allows easy management of user groups.

Managing Filesystems

A filesystem is a collection of files that allow a system to interact with these files.

These filesystems are accessed by special disks, such as /dev/dsk/c1d1s9, which would access the 9th partition on the 1st disk.

If you are familiar with Linux, it uses /dev/sd[l][n], where [l]is a letter and [n] is a number. This may look like /dev/sda1.

Adding Disks

UNIX makes it just as easy to mount and unmount disks. It's so easy, all you have to do is mount [file_location] [mount_point].
